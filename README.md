# 🐍 Snake Game - 贪吃蛇游戏

一个经典的贪吃蛇游戏，提供 **C++ 控制台版** 和 **HTML 网页版** 两种实现。

---

## 📂 项目结构

```
Snake-Game/
├── 贪吃蛇/                    # C++ 版本（Visual Studio 项目）
│   ├── snake.h               # 头文件（数据结构、函数声明）
│   ├── 源.cpp                # 主程序源码
│   └── 贪吃蛇.vcxproj        # VS 项目文件
├── snake_game(1).html        # HTML 网页版（AI 辅助生成）
├── 贪吃蛇.sln                # VS 解决方案文件
└── README.md
```

---

## 🎮 版本说明

### 1️⃣ C++ 控制台版（原生实现）

**技术栈：** C + Windows API

**功能特性：**
- ✅ 经典贪吃蛇玩法
- ✅ 用户注册 / 登录系统
- ✅ 最高分记录保存
- ✅ 排行榜（TOP 10）
- ✅ 数据持久化（本地文件）
- ✅ 随分数递增的游戏速度

**运行方式：**
1. 使用 Visual Studio 打开 `贪吃蛇.sln`
2. 编译并运行
3. 按菜单提示操作

**操作说明：**
- `W` - 向上
- `S` - 向下
- `A` - 向左
- `D` - 向右

---

### 2️⃣ HTML 网页版（🤖 AI 辅助生成）

> ⚠️ **声明：本版本由 GLM-5-Turbo 辅助生成**

**技术栈：** 纯前端（HTML + CSS + JavaScript），无后端依赖

**功能特性：**
- ✅ 现代化 UI 设计（暗色主题 + 霓虹风格）
- ✅ 用户注册 / 登录系统（localStorage 存储）
- ✅ 实时分数 + 最高记录显示
- ✅ 排行榜（TOP 10）
- ✅ 暂停功能（空格键）
- ✅ 移动端适配（触摸滑动 + 虚拟方向键）
- ✅ 响应式设计

**运行方式：**
- 直接用浏览器打开 `snake_game(1).html` 即可

**操作说明：**
| 按键 | 功能 |
|------|------|
| `W` / `↑` | 向上 |
| `S` / `↓` | 向下 |
| `A` / `←` | 向左 |
| `D` / `→` | 向右 |
| `空格` | 暂停 / 继续 |
| `ESC` | 返回菜单 |

**移动端：**
- 支持屏幕滑动控制
- 提供虚拟方向键按钮
