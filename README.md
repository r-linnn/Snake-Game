# Snake-Game 🐍
C语言实现的经典贪吃蛇小游戏

---

## 🎮 游戏介绍
这是一个基于C语言开发的控制台贪吃蛇游戏，复刻了经典玩法：
- 玩家通过方向键控制蛇的移动方向
- 吃到食物后，蛇身会变长，同时得分增加
- 撞到边界或自身身体时，游戏结束

## 🛠️ 运行环境
- 开发工具：Visual Studio 2022
- 编程语言：C语言
- 运行平台：Windows 系统（VS2022 编译运行）

## 🎯 操作说明
| 按键 | 功能 |
| :--- | :--- |
| `↑` `↓` `←` `→` | 控制蛇的移动方向 |
| 吃到食物 | 得分+1，蛇身变长 |
| 撞墙/撞自身 | 游戏结束 |
